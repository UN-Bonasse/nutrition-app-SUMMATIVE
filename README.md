Nutrition App

Nutrition App is a web application that provides real-time nutritional information for any food item using the API Ninjas Nutrition API. Users can search for foods and instantly get data on calories, protein, fat, carbohydrates, and serving size through a clean, responsive interface that works on desktop, tablet, and mobile. The backend, built with Node.js and Express, handles API requests and caches recent searches to reduce repeated calls, while the frontend uses HTML, CSS, and JavaScript to display results in a readable format. The app includes error handling for invalid inputs or API/network issues. To run the app, clone the repository, install dependencies with npm install, add your API key to a .env file, start the server with node server.js, and open http://localhost:3000 in a browser.
